## 操作方法

1. 选择阶段
   1. 输入坐标一，表示选中的棋子，进入操作阶段。
   2. 输入字母'w'，表示撤销。
   3. 输入字母'q'，表示结束游戏
2. 操作阶段
   1. 输入坐标二，表示将选中的棋子移到指定位置；
   2. 输入 "cancel"，表示取消行动。

## 胜负判断

将军死亡后，将死一方判负，另一方胜利。
